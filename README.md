# Görev Panosu

React, TypeScript, Vite ve Tailwind CSS ile hazırlanmış LocalStorage destekli görev yönetimi uygulaması.

Bu proje Web Geliştirme / JavaScript eğitim çıktıları için hazırlanmıştır. Uygulama modern bir JavaScript framework'ü kullanır, component yapısı içerir ve temel CRUD işlemlerini tarayıcı içinde çalıştırır.

## Özellikler

- Görev ekleme
- Görev listeleme
- Görev güncelleme
- Görev silme
- Durum ve öncelik filtresi
- LocalStorage ile tarayıcıda veri saklama
- Responsive arayüz

## Kullanılan Teknolojiler

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React
- LocalStorage
